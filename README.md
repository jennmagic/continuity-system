# The Continuity System

A drop-in instruction that keeps any AI project **resumable and accurate** — across long runs, new chat threads, or a switch to a different tool. It stops the two things that quietly wreck long projects:

1. **Hitting the wall** — the AI runs out of room and stalls mid-task.
2. **Fidelity loss** — the AI "summarizes" the conversation to save room and silently drops details, decisions, or reasoning.

The fix is simple: **important information lives in files, not in the chat.** This kit sets that up automatically.

---

## How to download

- **One file:** click the file below (e.g. `CLAUDE.md`), then click the **Download raw file** button (top-right of the file view).
- **Everything at once:** click the green **`Code`** button at the top of this page → **Download ZIP**.

---

## What you do (this is the whole job)

Pick the file named for your tool and put it in your project folder. **Nothing to rename.**

| Your tool | File to use |
|-----------|-------------|
| **Claude** (Claude Code / Cowork) | [`CLAUDE.md`](CLAUDE.md) |
| **Codex** or other AI coding tools | [`AGENTS.md`](AGENTS.md) |
| **Plain chat** (Claude.ai, ChatGPT) | [`CONTINUITY-SYSTEM.md`](CONTINUITY-SYSTEM.md) — paste it as your first message |

Then say: **"Set up the continuity system for this project."**

That's it. From then on the AI keeps everything current on its own.

---

## What it creates for you

Three small files that become your project's permanent memory:

| File | What it holds | Think of it as |
|------|---------------|----------------|
| **PROJECT.md** | The goal, scope, and rules | The north star |
| **STATE.md** | Where you are and the exact next step | The bookmark |
| **DECISIONS.md** | Every real decision and *why* | The memory of *why* |

You don't have to make these — the AI generates them on first run. Blank starter versions are in the [`templates`](templates) folder if you'd rather pre-place them.

---

## How to pick work back up (the payoff)

Open a new thread any time and say:

> **"Read PROJECT.md, STATE.md, and DECISIONS.md, then continue from the next action."**

The AI reloads the full picture from the files, not from a faded summary, and keeps going as if nothing was interrupted. No re-explaining.

---

## The one rule to remember

**If it matters, it goes in a file.** Everything else is just conversation, and conversation is disposable.
