# shared/ — finished artifacts, handed between apps

Put a component here only when it is **finished** and ready for the other app to use. Treat everything in this folder as **read-only** if you didn't produce it — copy it into your own workspace before changing it, don't edit it in place.

When you drop something here, add a one-line entry to `../DECISIONS.md` noting what was handed over and when, so the other app knows it's ready.
